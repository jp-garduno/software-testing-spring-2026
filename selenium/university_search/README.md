# University Search Automation

This project uses **Behave (BDD)** and **Selenium WebDriver** to automate Google searches and validate university-related results.

## 🧪 What it does

- Opens Google  
- Searches for a university  
- Clicks the first result  
- Validates the domain of the official page  
- Searches for careers related to the university  
- Checks if expected keywords appear in results  

## 🧰 Tech Stack

- Python  
- Selenium  
- Behave  

## 📁 Project Structure

- `features/` → Feature files (Gherkin scenarios)  
- `steps/` → Step definitions (Selenium code)   

## ▶️ How to run tests

- Install dependencies
- python3 -m behave
